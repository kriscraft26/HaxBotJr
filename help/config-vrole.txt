`config vrole {NAME} [-e|--entry VROLE] [-r|--remove ITEM] [-a|--add ITEM]`

Access the field `vrole.NAME`, the field can also be modified through the modifier flags.


Field with map value=
The value of `vrole` typed field is a map, with entries, and items under those entries.
If no entry is specified through `[-e|--entry VROLE]`, the modifier flags are applied to the entries,
if the entry is specified, the modifier flags are then applied to the items under that entry.


Configuration value=
All field access/modify commands has modifier flags that require a value to be followed after it,
the value can be in 2 forms:

__link/mention__: @role name, @user name, #channel-name
__name__: "role name", "user name", channel-name.

note if the name contains space, wrap it with `"`, for example "Top Gunner" instead of Top Gunner.


Examples=
`config vrole visual`
`config vrole visual -a "Top Gunner"`
`config vrole visual --remove @Top Gunner`
`config vrole visual -e @Top Gunner --add "Rocketeer"`
`config vrole personal --entry Princess -r @Princess Pontosaurus`
