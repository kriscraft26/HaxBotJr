`vote end <title> [-a|--anonymous]`

End currently active vote with specified title, and display the result.
If there title contains space, wrap it with `"`, for example `"vote title"` instead of `vote title`.


Anonymous=
If the anonymous flag is specified, the name of the voters for each options won't be displayed.


Examples=
`vote end "this is a vote"`
`vote end vote!!! -a`
